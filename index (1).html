Incentive Tracker — First time setup

Step 1: Google Apps Script Deploy karo

  1. Apni Google Sheet kholo aur Extensions > Apps Script click karo
  2. Code.gs file ka sara content delete karke download ki hui Code.gs file ka content paste karo
  3. setAdminPassword() function run karo ek baar (apna admin password set karne ke liye — default: manohar2025)
  4. initSheets() function run karo — ye saari required sheets automatically bana dega
  5. Deploy > New Deployment > Type: Web App > Execute as: Me > Who can access: Anyone > Deploy
  6. Deploy hone ke baad Web App URL copy karo (looks like: https://script.google.com/macros/s/AKfy.../exec)

Step 2: URL yahan paste karo

URL aapki browser mein locally save hogi — koi server nahi, data seedha Google Sheet mein
